
# Simpel aplikasi dengan spring boot

Aplikasi ini dibuat seperti layanan pada aplikasi Una Indonesia yaitu pada menu Asuransi Kecelakaan Diri

Untuk menjalankan aplikasi ini :

```bash
  docker compose up
```

Import SQL
```bash
  una.sql
```
